Unity Top Down Roguelike

Github Link To Project

Towards the end of my first year on the course we were assigned a group project, where students across the game programming course and the game art/design course would collaborate to create a project in either Unity or Unreal of our choosing. The project consisted of 3 programmers and 3 artists and each member was deligated different roles within the project. My tasks were to create the user interactables within the game as well as map design / construction, boss battle mechanics, player UI and small content scripts to add some depth.

C++ Snake Game

Github Link To Project

During my fundamentals of Programming module in year 1 I was tasked to recreate the popular game Snake, using only C++. This required me to apply my new knowledge of vectors and structs to parse game data through the various phases of the game - from initialisation to the completion stage. The project was worked on using industry standard Git version control. The game also takes advantage of storing user data from gameplay in external score files that can be loaded into a highscore table. I would consider this project complete but it is still on the back of my mind as I have numerous improvements I could make with my new C++ Experience since I began its' development.
